# Agentic UI

Welcome to **Agentic UI** - a comprehensive React component library for building AI applications with stunning interfaces.

## ğŸš€ Features

Agentic UI provides everything you need to build modern AI chat interfaces:

### Conversation Components
- **ChatMessage**: Role-based message bubbles with timestamps and avatars
- **MessageList**: Virtualized conversation history with auto-scroll
- **ChatInput**: Multi-line input with attachments and voice support

### Agent Management
- **AgentCard**: Profile cards with status indicators and capabilities
- **AgentSelector**: Dropdown for switching between agents
- **AgentGrid**: Responsive grid layout for multiple agents

### Streaming & Animation
- **StreamingText**: Character-by-character text animation
- **Typewriter**: Multi-word cycling animation
- **TypingIndicator**: Animated dots for thinking states
- **Loading States**: Skeletons, spinners, and pulse loaders
